# Poprawa jakości segmentacji semantycznej z wykorzystaniem kontekstu czasowego (WIP)

Projekt realizowany w ramach kursu Zaawansowane Sieci Neuronowe.

## Opis projektu

Głównym celem projektu jest zbadanie i implementacja metod poprawy jakości segmentacji semantycznej w sekwencjach wideo poprzez wykorzystanie informacji czasowej (temporal context). Tradycyjna segmentacja klatka-po-klatce często cierpi na brak spójności oraz błędy wynikające z chwilowych przesłonięć lub rozmycia ruchu. Nasze podejście zakłada wykorzystanie zależności między sąsiednimi klatkami w celu stabilizacji i poprawy dokładności masek segmentacji.
